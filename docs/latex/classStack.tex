\doxysubsection{Detailed Description}
A class that implements a \mbox{\hyperlink{classStack}{Stack}} using an array. 

This class provides a basic implementation of a \mbox{\hyperlink{classStack}{Stack}} data structure with fixed size using an array. It supports standard stack operations like push, pop, peek, and various utility methods.
